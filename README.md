Material for a data visualization workshop for the [Clarke Prize Environmental Design Challenge](https://ilead.engineering.utoronto.ca/scholarships-awards/clarke-prize/) (2026).  

Software requirement: [QGIS](https://qgis.org/download/)

Tutorial objectives:

1. Overview of QGIS (navigating a map, viewing base maps and data layers, and simple styling of data)

2. How to add in new layers and make a choropleth map (using PM2.5 air pollution data)

3. Show example of finding and downloading data to make a dot density map (using City of Toronto tree data)

4. Overview other data sources that might be useful for your projects.
