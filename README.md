Material for a data visualization workshop for Clarke Prize Environmental Design Challenge (2026).  https://ilead.engineering.utoronto.ca/scholarships-awards/clarke-prize/

Tutorial:

1. Overview of QGIS (base maps, data layers, and simple styling of pre-loaded data)

2. How to add in new layers and make a choropleth map (use PM2.5 air pollution data)

3. Show example of finding and downloading data to make a dot density map (City of Toronto tree data)

4. Overview other data sources that might be useful for your projects
